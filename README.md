# JS-DOM-demo
## 这是一个关于使用原生JavaScript对DOM/Css进行操作 和 针对JS事件模型进行模拟 的Demo
## 包含内容
### JS 操作DOM - Demo
 - 鼠标 穿越/离开 让图片 显现/消失
 - 使用按钮分别在首尾添加非空字符串
 - 点击每个元素li时在console调试台展示该元素文本内容
 - 按钮实现css效果切换

### JS 事件 - Demo
 - 模拟：冒泡 / 捕获
 - 取消事件进一步捕获或冒泡 / 取消事件默认行为
 - 模拟：自定义事件

### JS 动画 - Demo
 - 动画：小球移动<br>
   使用 requestAnimationFrame() 方法

### Tab 效果 - Demo
 - 使用 原生JS 实现 Tab 效果
 - 使用 jQuery 实现 Tab 效果
